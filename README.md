El ejercicio consiste en implementar un grafo en Java usando listas de adyacencia y aplicar los recorridos BFS y DFS para visitar los nodos en el orden correcto evitando ciclos. Para representarlo decidí usar un Map con implementación HashMap, porque me resulta más cómodo y claro trabajar con los nombres de los nodos directamente como claves y tener sus vecinos guardados en listas asociadas. De esta manera no dependo de índices numéricos y puedo acceder rápido a cualquier nodo, además de que las operaciones de inserción y búsqueda son más ágiles. En comparación con las listas que vimos en las conferencias, que se basan en posiciones y sirven más para explicar la teoría, el uso de Map y HashMap hace que la implementación sea más práctica y sencilla de manejar en código. En resumen mientras las listas ayudan a entender el concepto de adyacencia en papel, el Map me permitió llevar esa idea a Java de una forma más natural y fácil de aplicar
